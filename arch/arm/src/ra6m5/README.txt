This is a port of NuttX to the RA6M5 Family

Used board is the GLC23X PLC.

Most code is copied and adapted from the STM32U5 port.

The only supported RA6M5 family currently is:

-----------------------------------------------------------------
| NuttX config      | Manual | Chips
| RA6M5             | RM0456 | R7FA6M5BX
------------------------------------------------------------------
